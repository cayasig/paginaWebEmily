¿Por Qué Creamos Este Proyecto?

Nuestra Motivación

Como estudiantes del Colegio Dolores Sucre, hemos sido testigos de cómo muchos de nuestros compañeros enfrentan desafíos emocionales sin saber dónde buscar información o ayuda confiable.

Identificamos la necesidad de crear un espacio digital seguro, accesible y especialmente diseñado para adolescentes como nosotros, donde se pueda encontrar información clara sobre salud mental sin juicios ni estigmas.

Este proyecto representa nuestro compromiso con el bienestar de la comunidad estudiantil y nuestra contribución para normalizar las conversaciones sobre salud emocional entre los jóvenes.

Nuestros Objetivos

  • Crear un recurso educativo accesible para todos los estudiantes
  • Reducir el estigma asociado a la búsqueda de ayuda psicológica
  • Proporcionar información basada en fuentes confiables
  • Facilitar el acceso a los servicios de orientación del colegio
  • Promover una cultura de autocuidado emocional

Proceso de Desarrollo

1. Investigación: Recopilamos información sobre los principales trastornos emocionales que afectan a adolescentes de 15 a 17 años.

2. Planificación: Diseñamos la estructura del sitio web pensando en la experiencia del usuario adolescente.

3. Desarrollo: Utilizamos HTML, Bootstrap y JavaScript para crear un sitio web funcional y responsivo.

4. Diseño: Seleccionamos una paleta de colores cálidos (café/chocolate) para transmitir calma y confianza.

5. Contenido: Adaptamos la información a un lenguaje claro y cercano para nuestros compañeros estudiantes.

Nuestro Mensaje

"Creemos firmemente que la salud mental es tan importante como la salud física. Este proyecto es nuestro granito de arena para crear un entorno escolar más comprensivo, donde cada estudiante se sienta apoyado y sepa que no está solo en sus desafíos emocionales. Si este sitio ayuda a una sola persona a entender mejor sus emociones o a buscar ayuda, habremos cumplido nuestro objetivo."
